Spotify for Linux

Aqui você pode encontrar diferentes maneiras de instalar o Spotify para Linux. O Spotify para Linux é um trabalho de amor de nossos engenheiros que queriam ouvir o Spotify em suas máquinas de desenvolvimento Linux. Eles trabalham nele em seu tempo livre e atualmente não é uma plataforma que oferecemos suporte ativo. A experiência pode ser diferente de nossos outros clientes do Spotify Desktop, como Windows e Mac. Você pode nos dizer o que pensa e pedir ajuda a outros usuários no quadro Desktop (Linux) no fórum da Comunidade Spotify.

Install on Ubuntu

Vá para Spotify no Ubuntu Software e clique em instalar. Se o link não funcionar, abra o Ubuntu Software e pesquise por Spotify.

Install via command line

Snap

Se você não tem acesso ou não quer usar o Ubuntu Software, é possível instalar o Spotify a partir da linha de comando com snap. Execute o seguinte comando no seu terminal: 

snap install spotify
Se você executar outra distribuição Linux diferente do Ubuntu, primeiro consulte https://snapcraft.io/ para saber como instalar o snap e, em seguida, execute o comando acima.

Debian / Ubuntu

O Spotify para Linux também é lançado como um pacote Debian. Nosso objetivo é que ele funcione com a versão mais recente do Long Term Support do Ubuntu, mas tentaremos fazê-lo funcionar para outras versões do Ubuntu e Debian também.

Primeiro, você precisará configurar nosso repositório Debian: 

curl -sS https://download.spotify.com/debian/pubkey_6224F9941A8AA6D1.gpg | sudo gpg --dearmor --yes -o /etc/apt/trusted.gpg.d/spotify.gpg
echo "deb http://repository.spotify.com stable non-free" | sudo tee /etc/apt/sources.list.d/spotify.list

Então você pode instalar o cliente Spotify: 
sudo apt-get update && sudo apt-get install spotify-client
